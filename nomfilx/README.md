# Nomflix

Learning React and ES6 by building a Movie Discovery App.

## Screens

- Home
- TV Show
- Search
- Detail

## API Verbs

- Now playing (Movie)
- Top Rated (TV, Movie)
- Popular (TV, Movie)
- Upcoming (TV, Movie)
- Airing Today (TV)
- Search (TV, Movie)

## Code Challenges

- IMDB Link
- Tabs inside of Movie / Show Details (TV videos, Production Company & Countries)
- Collections Link
- Collections Route(/collection/{})
- On TV show, show season and creators
